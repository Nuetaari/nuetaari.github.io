abstract: >-
  An indexed prompt caching system designed to reduce the lookup latency of LLM responses using lightweight indexing schemes, including binary search trees (BSTs) by query length, top-n k-means clustering based on embeddings, and K-D trees. The experiment was done in Python (Jupyter Notebook).
